# Kelompok9-Praktik_AlgoritmaDanPemrograman

## Deskripsi
Sistem manajemen tugas untuk membantu mahasiswa mengelola tugas akademik dengan efisien. Sistem ini menggunakan algoritma *Selection Sort* untuk mengurutkan tugas berdasarkan deadline dan *Binary Search* untuk pencarian tugas tertentu.

## Tujuan
- Membantu mahasiswa dalam mengelola dan memantau tugas dengan pengurutan dan pencarian tugas berdasarkan deadline.
- Menghemat waktu pengerjaan tugas hingga 50% dibandingkan dengan cara manual.
- Sistem ini menggunakan struktur data sederhana seperti array dan record.

## Fitur Utama
1. **Input Tugas**: Menambah tugas baru dengan nama tugas, mata kuliah, deadline, dan status tugas.
2. **Tampilkan Semua Tugas**: Menampilkan daftar tugas yang sudah tercatat.
3. **Urutkan Deadline**: Mengurutkan tugas berdasarkan deadline dari yang terdekat.
4. **Cari Tugas**: Mencari tugas berdasarkan nama.
5. **Tugas Terdekat**: Menampilkan tugas dengan deadline terdekat.
6. **Ganti Status Tugas**: Mengubah status tugas menjadi selesai atau belum selesai.
7. **Menu Utama**: Menampilkan menu dengan pilihan untuk mengakses berbagai fitur.
